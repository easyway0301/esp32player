ESP32-S3 MicroPython 入門教學

第一步：認識 ESP32-S3 開發板

在開始之前，先來認識這塊 ESP32-S3 開發板上的兩個重要按鈕。

面向板子正面（USB 接口朝上）時：

  • 左邊為 RESET（重開機）按鈕：用來重新啟動板子。
  • 右邊為 BOOT 按鈕：主要用於進入燒錄（bootloader）模式。
圖片來源： Seeed Studio 官方文件

第二步：下載並安裝 Thonny

Thonny 是一款非常適合初學者的 MicroPython 開發環境。

請前往官方網站下載並安裝： https://thonny.org

第三步：將 ESP32-S3 連接電腦

使用 USB 傳輸線將 ESP32-S3 開發板連接到電腦。

Thonny 設定路徑：
執行 → 設定直譯器 → Thonny應該要用哪一種直譯器執行你的程式? → 選擇 MicroPython (ESP32)

請注意「選擇連接埠或是 WebREPL」的顯示狀態：

如果顯示為 Board CDC，代表目前不是燒錄模式。(如下圖)
如果顯示為 Serial debug unit，才是燒錄模式。(如下圖)
要進入燒錄模式，可依以下步驟操作： 按住 BOOT 按鈕不放 按一下 RESET 按鈕 最後放開 BOOT 按鈕 （按鈕會有清楚的「喀」一聲，正常使用不會按壞）

第四步：燒錄 MicroPython 韌體

依照下圖的設定，將 MicroPython 韌體燒錄至開發板中。

燒錄完成， 務必注意，要按一下Rest按鈕才能回到正常模式。
