TouchPad 觸控感測（GPIO 9）控制UserLed 實作

教學說明

本範例將示範如何在 Seeed XIAO ESP32-S3 上， 使用 TouchPad（電容觸控） 功能來偵測是否被人手或導線觸碰，以及控制UserLed燈。

本範例重點：
  • 使用 GPIO9 作為 TouchPad
  • 讀取原始觸控數值
  • 透過門檻值判斷是否「被觸碰」
  • 被觸控時打開UserLed
圖片來源： Seeed Studio 官方文件

MicroPython 範例程式

⚠️ 程式為無限迴圈執行，
若要停止程式，請在 Thonny 連按兩次 Ctrl + C 

from machine import TouchPad, Pin
import time

# 使用 GPIO9 的 Touch 功能
touch = TouchPad(Pin(9))

# 根據實測數值設定門檻
# 未觸碰：約 17331
# 觸  碰：約 147034
THRESHOLD = 50000

# 設定 LED 腳位
# 貼片式LED燈焊在GPIO 21 
UserLed = Pin(21, Pin.OUT)

def 打開UserLed():
    print("打開UserLed")
    # LED 亮燈
    UserLed.value(0)
    #UserLed.off() # 注意如果要用off，它會亮燈

def 關掉UserLed():
    print("關掉UserLed")
    # LED 熄燈
    UserLed.value(1)
    #UserLed.on() # 注意如果要用on，它會關燈

# 進入無限迴圈
# ⚠️ 若要停止程式，需在 Thonny 按兩次 Ctrl + C
while True:
    # 讀取觸控感測值
    # ⚠️ ESP32S3 的 Touch：
    #   - 沒碰時：數值通常較小
    #   - 觸碰時：數值會變得很大
    val = touch.read()
    
    # 將原始數值印出來，方便觀察與校正門檻
    print(val)

    # 判斷是否「被觸碰」
    # 門檻值需依實際環境調整（電線長度、濕度、人體）
    if val > THRESHOLD:
        print("⚡ GPIO1 被觸碰")
        打開UserLed()
    else:
        關掉UserLed()
        
    # 稍作延遲，避免刷太快
    time.sleep(0.1)
