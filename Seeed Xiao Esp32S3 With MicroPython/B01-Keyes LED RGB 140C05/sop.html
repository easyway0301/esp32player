KEYES 140C05 共陽極 RGB 全彩 LED

第一步：認識 RGB 共陽極 LED

本教學使用 KEYES 140C05 RGB 全彩 LED 模組， 這是一顆 共陽極（Common Anode） 的 RGB LED。

共陽極重點：
• 共用正極接 VCC
• 腳位輸出 0 = 亮燈
• 腳位輸出 1 = 熄燈
  • 紅色 LED → GPIO 9
  • 藍色 LED → GPIO 8
  • 綠色 LED → GPIO 7
圖片來源： Seeed Studio 官方文件

第二步：MicroPython 控制範例

以下程式會依序顯示 紅 → 綠 → 藍， 每個顏色亮 1 秒，再熄滅 0.5 秒。 


import machine, time

# 科易 KEYES LED 140C05 RGB 全彩 LED（共陽極）

# 共陽極 RGB LED 腳位
red = machine.Pin(9, machine.Pin.OUT)
blue = machine.Pin(8, machine.Pin.OUT)
green = machine.Pin(7, machine.Pin.OUT)

# 全部關燈（共陽極：1 = 關）
red.value(1)
green.value(1)
blue.value(1)

# 顏色清單（0 = 亮）
colors = [
    (0, 1, 1),  # 紅
    (1, 0, 1),  # 綠
    (1, 1, 0),  # 藍
]

for r, g, b in colors:
    red.value(r)
    green.value(g)
    blue.value(b)
    time.sleep(1)

    # 關燈
    red.value(1)
    green.value(1)
    blue.value(1)
    time.sleep(0.5)

print("結束了")

補充說明

如果你的 LED 行為「反過來」：
• 幾乎可以確定你用的是 共陰極 RGB LED
• 共陰極邏輯：1 = 亮、0 = 關

實際使用時，請依你手上的 LED 模組種類調整邏輯。
